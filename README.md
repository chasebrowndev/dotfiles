# dotfiles

Personal ricing configs for Hyprland + Kitty.

## Contents

- `.config/hypr/theme.conf` — Hyprland WM config (look, animations, keybinds)
- `.config/hypr/hyprland.conf` — sources theme.conf; machine-local overrides (input, monitors)
- `.config/kitty/kitty.conf` — Kitty terminal config (red/black theme, JetBrainsMono NF)

## Install

```bash
git clone git@github.com:chasebrowndev/dotfiles.git
cd dotfiles
cp -r .config/* ~/.config/
```
